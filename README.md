# Spam-Email-Detaection
A machine learning-based spam email classifier that detects whether an email is spam or ham (not spam) using NLP and classification algorithms. The model is trained on a labeled dataset and evaluated for accuracy using various performance metrics.  Tech Stack: Python, Pandas, Scikit-Learn, NLTK
